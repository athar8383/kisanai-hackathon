# KisanAI Hackathon

Agentic agricultural decision support app for Pakistan.

## Features
- Mobile-friendly web app
- AI-assisted advisory workflow
- Deployable with Docker / Cloud Run

## Run locally
Open the app files in a static web environment or deploy with the included Dockerfile.

## Deployment
This project was deployed using Google Cloud Run.

## Notes
Users provide their own API key at runtime if required. No API key is bundled in the repository.
